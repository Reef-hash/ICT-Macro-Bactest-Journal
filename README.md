## 🛠️ Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - CSS Variables, Flexbox, Grid, Backdrop-filter
- **JavaScript (ES6+)** - Vanilla JS, LocalStorage API
- **No Framework** - Pure vanilla untuk performance & portability

## 📱 Cara Guna

1. **Download** atau **clone** repository
2. **Buka** `index.html` dalam browser (Chrome/Safari/Edge)
3. **Masukkan trade:**
   - Risk (pip/points)
   - Reward (pip/points)
   - Outcome (WIN/BE/LOSS)
4. **Ulang** sehingga 300 trade
5. **Export CSV** untuk backup

## ⚠️ Nota Penting

**Data disimpan secara lokal di browser sahaja (LocalStorage):**
- ✅ Data kekal walaupun tutup & buka semula
- ❌ Tidak sync antara peranti (Telefon ≠ PC)
- ❌ Hilang jika padam cache/browser data
- 💡 **Export CSV secara berkala untuk backup**

## 🔒 Privacy

100% client-side. Tiada data dihantar ke server. Semua kiraan & storage berlaku dalam browser anda.
