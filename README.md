# 📊 ICT Macro Journal

A modern **ICT-based trading journal** built for backtesting and forward testing.  
Designed to track performance using **Risk-Reward (RRR), expectancy, drawdown, and prop firm metrics** — all in a clean, interactive UI.

---

## 🚀 Features

- 📈 **Trade Tracking**
  - Record Risk, Reward, Outcome (Win / BE / Loss)
  - Auto RRR calculation

- 📊 **Advanced Statistics**
  - Winrate
  - Expectancy (R)
  - Total PNL (Points & USD)
  - Max Drawdown
  - Win/Loss Streak
  - Avg Monthly Return

- 🧠 **Prop Firm Evaluation**
  - Auto-check eligibility based on:
    - Winrate ≥ 55%
    - Drawdown ≤ 3%
    - Monthly Return ≥ 6%

- 📉 **RRR Analysis**
  - Breakdown by:
    - 1:1
    - 1:2
    - 1:3+
  - Profitability indicator

- 📅 **Trading Calendar**
  - Visual heatmap of trading performance
  - Daily breakdown (Win / Loss / BE + PNL)

- 💾 **Local Storage System**
  - All data stored in browser (no account required)

- 📥 **CSV Export**
  - Full trade history + summary stats
  - Backup your data anytime

## 🧑‍💻 How To Use

1. Enter:
   - Risk (points)
   - Reward (points)

2. Select outcome:
   - ✅ Win
   - ➖ Break Even
   - ❌ Loss

3. Click **➕ Add Trade**

4. View:
   - Stats update automatically
   - Calendar performance
   - RRR breakdown

5. Backup data:
   - Click **📥 Download CSV**

## ⚠️ Important Notes

- This is a **static website**
- Data is stored using **localStorage**
- Your data:
  - ❌ Not synced across devices
  - ❌ Will be lost if browser cache is cleared

👉 Always export your data using CSV for backup
